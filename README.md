# Algoritmos y Programación 1 — Material de la materia (UNTreF)

Este repositorio contiene el material de la cátedra *Algoritmos y Programación 1* de la UNTreF. Acá vas a encontrar guías, enunciados, ejemplos y códigos usados a lo largo del cuatrimestre.

## Estructura del repositorio

```
.
├── examenes — Enunciados y resoluciones de exámenes.
├── greenfoot — Proyectos y ejemplos en Greenfoot (Java).
├── guias — Ejercicios de clase.
├── notebooks — Notebooks con ejemplos y análisis.
└── trabajos-practicos — Enunciados de los TP.
```

**Nota:** si ves alguna carpeta que falta describir, abrí un [issue](https://github.com/untref-ayp1/repo/issues) o mandá un PR con la aclaración.

## Cómo contribuir

- Trabajá en una rama separada.
- Hacé commits claros y atómicos.
- Subí tus cambios y abrí un pull request.
- Si sólo necesitás corregir un README o dejar una sugerencia, podés abrir un [issue](https://github.com/untref-ayp1/repo/issues).

## Recomendación de actualización

Se sugiere sincronizar frecuentemente la copia local con los cambios del curso. Al menos hacé pull (actualizá tu copia local) al menos una vez por semana, ya que durante el cuatrimestre se harán cambios frecuentes.

## Contacto / Soporte

Abrí un [issue](https://github.com/untref-ayp1/repo/issues) en el repo para reportar errores, faltantes o dudas. También podés comunicárselos al ayudante o docente de la cátedra.
